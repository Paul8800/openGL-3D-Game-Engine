# OpenGL 3D Game Engine

## Features
Implimented:
- Collision
- Model loading
- 2D GUI

Working on:
- Lighting
- Audio (mic input, positional audio, music, sound effects)
- Animations
- Particles
- AI Pathfinding
- Mesh Culling
- Save States
- Networking Multiplayer

### Collision detection:
- Sphere
- AABB
- OBB
- Convex Hull (in progress)
- Triangle Collision (in progress)

- Raycasting (in progress)
- Capsule (in progress)
- Heightmap (in progress)
